# Homebrew Tap for Intent-Engine

This is the official Homebrew tap for [Intent-Engine](https://github.com/wayfind/intent-engine).

## Installation

```bash
# Add the tap
brew tap wayfind/tap

# Install intent-engine
brew install intent-engine

# Verify installation
ie --version
ie doctor
```

## Update

```bash
brew update
brew upgrade intent-engine
```

## Uninstall

```bash
brew uninstall intent-engine
brew untap wayfind/tap
```

## What is Intent-Engine?

Intent-Engine (`ie`) is an AI long-term task memory system that provides:

- **Cross-session persistence** - Tasks survive across multiple AI sessions
- **Hierarchical task management** - Break down complex tasks into subtasks
- **Decision records** - Track why decisions were made
- **Focus-driven workflow** - One focused task at a time
